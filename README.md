# Salarite Virtual HR

Full Stack Virtual HR & ATS Dashboard System

## Tech Stack
- Backend: FastAPI
- Database: SQLite
- ORM: SQLAlchemy
- Frontend: React
- API Integration: Axios

## Features
- Business Dashboard
- HR Dashboard
- Employee Dashboard
- Task Assignment & Tracking
- Interview Scheduling
- Google Meet & Phone Call Tracking
- Live Statistics

## How To Run

### Backend
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

### Frontend
cd frontend
npm install
npm start
